OCR Project in C: Collaborative Achievement

In a collaborative effort, our team successfully developed an Optical Character Recognition (OCR) system using the C programming language. Notably, we built this OCR from scratch without relying on external libraries.

Key Highlights:

  • Image Processing: Implemented advanced techniques for image preprocessing.
  • Character Segmentation: Achieved precise segmentation of characters from images.
  • Pattern Matching: Utilized custom algorithms for accurate character recognition.

Challenges Overcome:

  • Algorithmic Complexity: Managed complexities in designing algorithms for diverse fonts and styles.
  • Resource Optimization: Ensured efficient memory usage for improved performance.

Outcome:

Our team's dedication resulted in a functional OCR system capable of accurately recognizing text from various images. This project not only enhanced our programming skills but also provided valuable insights into computer vision and image processing.

A Long Night - A Unity Horror Game

During our Unity game development project, my team and I created an immersive horror experience called "Deadly Escape." The game challenges players to navigate through a haunted house while avoiding a relentless killer. One of the most exciting features is the multiplayer option, adding a thrilling social dimension to the horror.

Game Overview:

"Deadly Escape" immerses players in a chilling atmosphere with carefully crafted graphics and a suspenseful soundtrack. Each player takes on the role of a character trapped in a sinister house, tasked with the sole objective of escaping alive.

The unpredictable killer patrols the house, creating constant tension. Players must use their ingenuity to find clues, solve puzzles, and unlock secret passages while avoiding the killer. Cooperation between players is crucial for survival, adding a strategic element to the game.

Multiplayer Features:

What sets "Deadly Escape" apart is the ability to play in multiplayer mode. Players can team up, strategize, and escape the killer together. Communication is key as coordinated movements and actions are vital for success.

Multiplayer sessions offer a variety of tactics, from diversion to coordinated escape routes. This creates a dynamic and interactive gaming experience, where each session is unique based on player choices and actions.

Conclusion:

"Deadly Escape" goes beyond a typical Unity horror game in C#. It represents the culmination of a passionate team's work, dedicated to delivering a memorable experience. With its eerie atmosphere, challenging puzzles, and captivating multiplayer mode, the game promises thrills and suspense for those daring enough to enter this virtual haunted house.
